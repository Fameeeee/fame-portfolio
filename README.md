# Portfolio Website

This is a personal portfolio starter built with Next.js and prepared for Vercel deployment.

## Customize

Update these files with your real information:

- `app/page.tsx` for your name, projects, experience, and links
- `app/layout.tsx` for the browser title and description
- `app/globals.css` for colors, spacing, and visual style

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Sign in to [Vercel](https://vercel.com).
3. Click **Add New Project**.
4. Import your GitHub repository.
5. Keep the default Next.js settings and deploy.

Vercel will detect the framework automatically.
